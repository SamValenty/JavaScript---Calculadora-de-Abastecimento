# JavaScript---Calculadora-de-Abastecimento
Este projeto é uma calculadora de abastecimento desenvolvida com foco em JavaScript moderno. O sistema permite selecionar o tipo de combustível e calcular automaticamente o valor total com base na quantidade informada.
O código foi desenvolvido usando arrow functions, deixando tudo mais moderno e fácil de entender. Também inclui validações para evitar erros, como campos vazios ou valores inválidos, além de formatar o resultado no padrão brasileiro (R$). A atualização acontece em tempo real, o que deixa a experiência mais dinâmica e fluida.
